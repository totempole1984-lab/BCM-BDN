StrategyTrace Demo Dataset (POS)

Files:
- capabilities_l1.csv — L1 capability map (generic).
- drivers_benefits.csv — drivers with attached benefits and metrics.
- changes.csv — business-facing changes linked to benefits and capabilities.
- enablers_mock_jira.csv — mock Jira Epics (POS) with status & points.
- mapping_enablers_changes.csv — links Epics (Enablers) to Changes with weights.

How to import (suggested order):
1) capabilities_l1.csv
2) drivers_benefits.csv
3) changes.csv
4) enablers_mock_jira.csv
5) mapping_enablers_changes.csv
